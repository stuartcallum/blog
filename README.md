# Callum's Travel & Dev Blog

## Posts
To create a new post, run the following command:
```
hugo new posts/my-first-post.md
```
Then, edit the `my-first-post.md` file to suit your needs.

You can view your changes with
```
hugo server -D
```
When happy with your post, change `Draft: true` to `false` in the header of your `my-first-post.md` file. 


## Markdown Syntax
Please view `./content/posts/my-first-post.md` for examples.
